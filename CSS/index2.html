KARA

Kara es un androide modelo AX400, creado para encargarse de las tareas del hogar y el cuidado de niños.
Esta asistente del hogar es capaz de hablar 300 idiomas y puede cocinar unas 9.000 recetas diferentes.
Ahora trabaja para Todd, un taxista con problemas con el alcohol, y su hija pequeña, Alice.
